# IndexV2QC
A version of IndexV2 with support for multiple titles.

Index is a tool for Halo: Combat Evolved Anniversary and Halo 2 Anniversary and Quake Champions that extracts textures, models, and more.

![screen](https://github.com/StremaLot/IndexV2_QC/blob/game-modules/Awoken.jpg)

Please don't hate me much. I'm total newbee in MODs, in C# and in programming as well. For newbees like me instructions below.
Installing:
1. Download Visual Studio (Im using 2022).
2. On start of VS clone github repository with paste a link  https://github.com/StremaLot/IndexV2_QC.git.
3. Add existing progect LibSaber to Index.App via right click on it at solution explorer.
4. Right click on Index.App and Publish.
